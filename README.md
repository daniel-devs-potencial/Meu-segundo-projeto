Aplicação web desenvolvida em **HTML**, **CSS** e **JavaScript** para converter valores de Real para outras moedas. O projeto ilustra conceitos básicos e intermediários de manipulação de DOM e integração de lógica de conversão de moedas.

## 💻 Sobre o Projeto

Este projeto oferece uma interface simples e prática para conversão de moedas. O usuário pode digitar um valor em real (BRL) e escolher para qual moeda deseja converter: Dólar Americano (USD), Euro (EUR) ou Libra Esterlina (GBP). O resultado aparece automaticamente na tela, utilizando formatação local para cada tipo de moeda.

## ✨ Funcionalidades

- Seleção da moeda de origem (Real) e destino (Dólar, Euro, Libra).
- Conversão instantânea ao clicar em "Converter".
- Atualização do nome e da bandeira da moeda destino automaticamente.
- Formatação internacional dos valores exibidos para melhor experiência do usuário.

## 🖼️ Interface

A interface inclui:
- Seletores de moedas (com ícones e nomes).
- Campo de entrada para o valor em reais.
- Botão de conversão.
- Exibição visual do valor convertido com nome e ícone da moeda correspondente.

## 🚀 Como utilizar

1. Clone o repositório:
    ```bash
    git clone https://github.com/daniel-devs-potencial/Meu-segundo-projeto.git
    cd Meu-segundo-projeto
    ```

2. Abra o arquivo `index.html` no seu navegador preferido.

Pronto! Basta digitar o valor, escolher a moeda e clicar em "Converter".

## 📁 Estrutura de Arquivos

- `index.html` — estrutura da página e elementos do conversor.
- `css/` — estilos utilizados para layout e design.
- `scripts.js` — lógica da conversão e manipulação da interface.
- `assets/` — imagens dos ícones das moedas e do projeto.

## 🛠️ Tecnologias

- HTML5
